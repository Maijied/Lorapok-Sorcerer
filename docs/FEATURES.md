# Features

- Original-image discovery through the first-party LoraMediaFinder Engine.
- Powered by a standalone normalize → rules → score → dedupe pipeline.
- Twitter, Wikimedia, Pinterest, Googleusercontent, and generic resize URL
  cleanup.
- `srcset`, `<picture>`, lazy-load attributes, wrapping links, and page
  metadata support.
- Candidate verification by media content type and download size.
- Image and video attachment uploads up to Discord's 10 MB fallback threshold.
- URL-only fallback when an attachment cannot be fetched or is too large.
- Preview workflow with candidate chips, LoraMediaFinder badge, zoomable image/video,
  note/links field, character count, channel selector, and keyboard send.
- Quick-send mode for image context menus.
- Context menus for images, links, and selected text.
- Toolbar popup for text/URL sending.
- Multiple Discord webhooks with validation, enable/disable controls, testing,
  and channel avatars.
- Options tabs for Channels, Sending, History, and About.
- Recent-send history retained in local storage (up to 50 entries).
- Firefox MV2 and Chromium MV3 builds for Chrome, Edge, Opera, and Brave.
- No remote images, scripts, or fonts in extension pages.
